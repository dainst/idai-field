# Notice

This repository is derived from iDAI.field, an archaeological field documentation
system developed by the German Archaeological Institute (DAI) and GBV.

- Original repository: https://github.com/dainst/idai-field
- Project website: https://field.idai.world/
- License: Apache License 2.0

The Korean Field Notebook work in this repository adapts that codebase for
Korean archaeological field recording workflows, including Korean project
templates, feature candidate recording, map-based sketching concepts, pen memo
records, aerial background layer support, and soil profile photo/soil color
recording.

Relevant reference:

S. Hohl, T. Kleinke, F. Riebschlaeger, J. Watson, "iDAI.field: developing
software for the documentation of archaeological fieldwork", Archeologia e
Calcolatori 34.1, 2023, 85-94. DOI: https://doi.org/10.19282/ac.34.1.2023.10

보존하면서 한국 매장문화재 조사 현장에 맞는 기록 흐름을 별도로 발전시키는
것을 목표로 합니다.
