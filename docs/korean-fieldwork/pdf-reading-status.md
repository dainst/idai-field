## 현재 판단

10종 모두 로컬 PDF 확인, 텍스트 추출, 권별 노트 반영까지 완료했다. 다만 이것은 “야장 모델 설계에 필요한 1차 정밀 독해” 상태이며, 실제 iDAI.field 구성 파일을 고치기 전에는 다음 보강이 더 필요하다.

우선순위:

1. 제7권 성곽은 판축·읍성·왜성·보존관리 장의 야장 항목을 보강했고, 시설별 제원도 수원화성·남한산성 사례 중심으로 1차 정밀화했다. 이번 재대조에서 사례 지명 색인과 도표 제목 교정을 한 차례 더 진행해 성곽도면색인, 문헌척도환산, 읍성규모변경, 방어시설개편, 영남성곽사례색인, 왜성입지구조, 화성공심돈제원 값을 추가했다. 실제 구현 전에는 원문 PDF 쪽 깨진 한자와 개별 도판 번호를 다시 대조한다. 제5권 고분은 석곽·석실 조사절차에 더해 목관·목곽, 봉토 조사통제, 적석목곽, 옹관, 인골 잔존·수습·보존처리·복원·보고 항목을 추가 보강했고, 이번 재대조에서 도표·사례 색인, 분구묘 단계판정, 옹관분구축조관계, 부장품 기능 맥락, 외래 묘제 속성대조까지 1차로 반영했다. 남은 일은 OCR이 흔든 한자·개별 캡션·도판 번호를 원 PDF 화면과 다시 대조하는 것이다.
2. 제1권 총론은 조사운영, 정보화, 보존·정비 원칙을 실제 템플릿 우선순위에 맞춰 보강했고, 텍스트 추출이 깨졌던 건물터·고분 장도 Windows OCR로 1차 회수했다. 이번 보강에서 보존가치 세부기준, 고유성·보전성 요소, 보존관리위협, 재매장·새재료 구분·기록요건, 정보화 추진주체·활동영역·전자보고서 산출물을 추가했다. 남은 작업은 한자·도판 캡션·사례 수치 교정과 제3권·제5권 세부 필드와의 최종 대조다.
3. 제2권은 구석기 편년 판단, 제4기 지질·고토양·토양쐐기, 석기 재질 표준화, 현재 내륙화된 패총 입지 가능성을 재독해 보강했고, 구석기 앞 장을 다시 대조해 층위 재검토와 사용흔 분석 후보 석기 처리까지 추가했다. 남은 작업은 깨진 한자·지명·도표 제목과 사례별 세부 수치 교정이다.
4. 제3권 건물지는 OCR 전체 재독해를 마치고 한국형 야장 항목을 확장했다. 제6권은 충적지 층서의 a/b 세트와 유구면·검출면 판정을 추가 보강했다. 남은 작업은 깨진 한자·지명·도표 제목과 사례별 수치 교정이다. 제5·6권 OCR 자료도 필요한 범위에서 같은 방식으로 교정한다.
5. 권별 노트에서 뽑은 값 목록을 `core/config/Library/Valuelists` 쪽 실제 한국형 구성으로 옮긴다.
6. 샘플 조사 프로젝트를 만들어 고분, 패총, 제철유적 각각의 입력 흐름을 검증한다.
