# 한국형 야장 기록 워크플로 초안

이 문서는 『한국 매장문화재 조사연구방법론』 독해에서 뽑은 관찰 항목을 실제 야장 화면 흐름으로 바꾸기 위한 중간 설계다. 원문 요약이 아니라 iDAI.field를 한국 현장용으로 고칠 때 필요한 작업 순서와 기록 단위를 정리한다.

## 1. 공통 흐름

모든 유적 유형은 다음 기록 흐름을 기본값으로 둔다.

1. 오늘의 작업기록을 만든다.
2. 조사구역, 그리드, 트렌치, 기준점을 연결한다.
3. 관찰 대상이 층, 유구, 유물, 시료, 보존조치, 매체 중 무엇인지 고른다.
4. 관찰 사실을 먼저 적고, 해석과 해석 근거는 분리한다.
5. 사진, 도면, 3D, GIS, 시료, 수습품을 같은 기록 ID에 연결한다.
6. 불확실성, 대안 해석, 확인 필요 작업을 남긴다.
7. 보고서 반영 여부와 수정 이력을 남긴다.

이 흐름은 “한 번 입력하고 끝나는 카드”가 아니라 현장에서 계속 갱신되는 기록이어야 한다. 예를 들어 유구선을 처음 그은 뒤 토층 판단이 바뀌면, 기존 기록을 덮어쓰기보다 수정 전/후 판단과 근거를 함께 남긴다.

## 2. 충적지와 층위 중심 유적

충적지 유적은 평면 유구 입력보다 단면과 층 해석, 미지형 입지모델이 먼저다.

1. 지표조사 단계에서 문헌, 고지도, 지형도, 위성영상, GPS/GIS 관찰, 주변 하천과 기후 정보를 연결한다.
2. 표면 유물이 없는 평야·논 지역도 고지형 분석 대상으로 남긴다.
3. 위성사진, 여러 시기의 항공사진, 고지도, 지형도, 지질도, DEM, 형질변경 지도, 시추·지반자료를 자료 목록으로 묶는다.
4. 지형역환경, 지형면환경, 지형대환경, 미지형환경, 극미지형환경 중 어떤 분석 단계인지 표시한다.
5. 자연제방 전사면·상면·후사면, 배후습지, 구하도, 선상지, 단구, 미고지 같은 미지형 후보를 세운다.
6. 현지답사로 인위적 형질변경과 판독 오류를 보정하고, 최종 고지형 분석도와 시굴 제안 범위를 만든다.
7. 기준점과 격자를 설치하고, 조사구역 밖 안정 기준점과 격자 방향의 근거를 기록한다.
8. 시굴 트렌치는 미지형 방향과 격자를 기준으로 배치하고, 가능성이 높은 중심 지점과 양 끝 지점을 우선 확인한다.
9. 각 트렌치에서 단면 위치, 기준점, 고도, 촬영 방향, 층위, 유구 흔적을 기록하고 전체 주상도로 묶는다.
10. 층을 나누기 전에 어떤 층 개념을 적용하는지 정한다.
11. 색, 입도, 점성, 혼합상태, 포함물, 경계, 연속성, 산화철·망간, 지하수 영향, 생물교란을 기록한다.
12. 색 차이만으로 층을 나누었는지, 구성 물질과 퇴적구조가 실제로 달라지는지 검토한다.
13. 산화철·망간, 지하수위 변동, 뿌리·말목 주변 변색 같은 2차 현상은 층과 별도로 표시한다.
14. 문화층, 자연퇴적층, 토양층, 경작층, 검출면, 기능면 중 어떤 판단인지 적는다.
15. 건층 후보와 층간 대비, 신구관계를 기록한다.
16. 장비 제토의 목표 심도, 잔여 두께, 장비 동선, 조사원 입회자, 유구 확인 방식을 남긴다.
17. 평면 조사를 시작할 대상층과 그 이유를 남긴다.
18. 유구의 축조면, 사용면, 검출면, 소속층위, 기반층, 내부토를 분리해 기록한다.
19. 유구면이 하위층에서 확인되더라도 상위 토양화층에서 굴착되었을 가능성과 상위층 두께를 함께 계산한다.
20. 주거, 무덤, 밭, 논, 야외노지, 수혈, 주혈, 도로, 의례유구가 미지형 안에서 어떤 관계를 갖는지 연결한다.
21. 논·밭은 기능면, 가공면, 두둑·고랑, 물관리, 경작구흔, 식물규산체, 토양미세형태 시료를 함께 기록한다.
22. 유구와 유물이 어느 퇴적 단위에 속하는지 연결한다.
23. 유구 번호를 축으로 유물, 사진, 도면, 조사기록, 행정서류 정리 상태를 계속 점검한다.
24. 자연과학 분석과 연대측정은 시료 종류보다 분석 질문, 선택 이유, 유구 간 관계 검토 목적을 먼저 남긴다.
25. 고환경 분석은 집수유역, 광역·주변·미세지형, 단면·시추 시료, 분석법, 절대연대, 복원 차원을 하나의 흐름으로 묶는다.
26. 화분·규조·유공충·패류·식물규산체처럼 보존 조건과 운반 과정에 민감한 자료는 원지성/이지성, 산화·경작 영향, 분석 한계를 함께 기록한다.

야장 화면에서는 단면 사진과 스케치, 층 설명, 시료 번호, 고도값, 주상도, 제토 기록이 한 화면에 같이 보여야 한다. 층 이름만 입력하게 하면 충적지 조사의 핵심 정보가 빠진다.

## 3. 선사·석기 유적

선사 유적은 산포와 퇴적 후 변형을 함께 다룬다.

1. 유물 출토 위치, 좌표, 높이, 층위, 방향성을 기록한다.
2. 산포 밀도와 수습 방식을 남긴다.
3. 홍수, 침식, 재퇴적, 생물교란, 뿌리 교란, 화학적 변질 가능성을 검토한다.
4. 문화현상으로 판단한 근거와 자연현상 가능성을 분리한다.
5. 석기인지 판단하기 어려운 자료도 후보석기로 남기고 3차원 위치를 보존한다.
6. 석기는 원재료, 피질, 박리면, 타격점, 제작기법, 잔손질, 작업날, 등손질, 사용흔을 기록한다.
7. 접합군은 조각별 좌표와 수직 분포를 연결하고, 제작장 해석은 미세 격지와 체질 결과를 함께 본다.
8. 석기 도면은 타격면, 선단부, 배면·복면, 박리면, 능선, 작업날 표시 상태를 검토한다.
9. 기본 수습 카드와 정밀 석기 분석 카드를 분리하되 같은 식별자로 연결한다.

석기 형식명은 드롭다운의 첫 입력값이 아니라 관찰값을 바탕으로 나중에 확정되는 해석값에 가깝다. 현장에서는 “긁개”라고 적는 것보다 왜 그렇게 보았는지를 남기는 편이 더 중요하다.

## 4. 패총과 동물유체

패총은 생업 자료이면서 퇴적 과정이 복잡한 층위 자료다.

1. 패각층의 범위, 두께, 패각 종류, 퇴적물 구성을 기록한다.
2. 사면 이동, 해수면 변동, 해안 모래 유입, 패각 붕괴와 재퇴적 가능성을 검토한다.
3. 공반 유물은 퇴적 과정 검토 후 같은 행위 단위로 묶는다.
4. 동물유체는 큰 자료만 선별하지 않고 체질 또는 표본 채취 방식을 기록한다.
5. 마른 체질, 물체질, 부유법, 전체 체질, 표본 체질 중 무엇을 사용했는지 남긴다.
6. 표본 채취량, 채취 층위, 분석 목적, 전문가 확인 상태를 기록한다.

야장에는 체질하지 않은 자료와 체질한 자료가 같은 신뢰도를 갖지 않도록 수습 방식 필드가 필요하다.

## 5. 건물지

건물지는 구조 요소, 생활공간, 유물 원위치, 복원 가설을 분리해서 쌓는다.

1. 상위 건물지 기록을 만든다.
2. 집자리와 집터를 구분하고, 건물 밖 생활공간과 외부 유물 산포 범위를 함께 잡는다.
3. 주공, 초석, 기단, 벽체, 바닥, 배수, 화덕, 출입부, 저장·침상 시설을 요소 카드로 추가한다.
4. 각 요소의 위치, 크기, 깊이, 재료, 보존 상태, 절단 관계를 기록한다.
5. 바닥면 유물은 수습 전 원위치, 높이, 방향, 요소 주변 위치, 유물군 배열, 사진·도면 번호를 남긴다.
6. 기단 유형과 기초공법을 별도 카드로 기록한다.
7. 초석, 적심, 기단석, 판축층, 되파기, 교란층이 어느 건물 단계에 속하는지 판단한다.
8. 축조, 개수, 사용, 소실, 폐기 단계를 따로 기록한다.
9. 주공이 구조용인지 비구조용인지 판단하고, 어깨선, 도리선, 서까래 밑동 후보, 마루기둥 후보를 표시한다.
10. 정면/측면 칸수, 주간거리, 어칸·협칸·퇴칸, 기단 총장, 초석열 총장으로 영조척 후보를 계산하고 오차 범위를 남긴다.
11. 결구방식, 내진/외진, 내진주/고주, 감주 여부, 보·도리·서까래·공포 같은 목조건축 구조 후보를 입력한다.
12. 지붕 구조, 지붕재, 벽체, 출입 방식, 배연·채광, 난방, 생활 동선, 저장시설 같은 복원 가설을 복수로 남긴다.
13. 발굴유구, 문헌, 옛 사진, 비교건물, 전문가 자문, 시공 중 변경사항을 복원 근거로 연결한다.
14. 기와와 건축부재는 대표품만 뽑지 말고 층위·그리드·건물별 출토 위치와 속성 조사 범위를 기록한다.
15. 수키와/암키와, 문양, 제작기법, 내면·외면, 단면, 타날구, 소지, 잔존 상태를 목적에 맞는 속성표로 남긴다.
16. 초석이 없어진 곳에서는 초석설치굴광, 적심석, 보강토, 초석발취공, 초석폐기공, 지하식 초석을 확인한다.
17. 기단외장, 계단, 낙수구, 처마 돌출도, 비계 주공, 기준 말뚝처럼 건물 주변 미세 유구를 구조 요소와 분리해 기록한다.
18. 중복 주거지의 동시성, 개축·중수, 장기 형성 가능성을 별도 판단값으로 남긴다.
19. 유구선 수정 이력, 제토 방식, 토층 판단자, 용어 선택 근거를 기록한다.
20. 노출 완료 뒤 사진·도면·야장기록을 마친 다음, 상면에서 하부로 층위와 깊이를 보존하며 수습한다.
21. 수습 뒤에는 필요한 범위의 단면조사로 하부구조와 구축방법, 하부 유구 존재를 확인한다.

건물지 화면은 “건물지 개요 + 요소 목록 + 원위치 유물 분포 + 건축부재 속성표 + 단계 다이어그램 + 영조척 계산 + 복원안 비교” 구조가 적합하다. 평면 노출만으로 끝내면 건축 기능과 생활상을 복원하기 어렵다.

## 6. 고분

고분은 무덤 단위, 구조 단위, 매장 이벤트, 수습 단위를 계층화한다.

1. 고분 기본 정보와 주변 고분 관계를 기록한다.
2. 봉토 트렌치 설정 전 묘도·연도 후보 위치를 검토한다.
3. 봉토의 성토 재료, 성토 순서, 구지표 정지면, 호석, 경사각, 선후관계를 기록한다.
4. 주구의 굴착 범위, 평면·단면 형태, 인접 주구와의 중복/공유, 내부 퇴적토와 포함 유물을 기록한다.
5. 제사유구는 위치, 유물 상태, 바닥시설, 추가매장과의 관계를 남긴다.
6. 묘도·연도는 평면 조사 뒤 최소 파괴 트렌치로 단면을 대조한다.
7. 폐쇄석은 재질, 크기, 방향, 놓인 순서, 제거 순서, 추가매장 흔적을 기록한다.
8. 석실 개방 전 밀폐 상태, 내부 온습도, 공기·토양 시료, 개방 시각을 기록한다.
9. 개방 후 내부환경 변화, 부장품 위치, 인골 자세와 보존 상태를 기록한다.
10. 부장품은 관 내부, 관 주변, 별도 부장공간, 현실, 연도, 묘도, 주구, 분구 같은 위치군으로 묶고 높이와 배열을 남긴다.
11. 부장품 성격은 착장품, 위신·의장용품, 무기·마구·농공구, 공헌제사품, 벽사품, 명기, 외래계 유물 후보로 분리한다.
12. 제사자료는 축조 공정 단계, 부장/공헌 구분, 토기 파손, 음식물, 동물희생, 소토·목탄, 제단·수혈 여부를 기록한다.
13. 인골은 노출 도구, 뿌리 제거, 직사광선 차단, 물 사용 여부, DNA 시료 선채취 여부를 남긴다.
14. 인골은 1/5 도면, 세부 사진, 포장재, 비닐 밀봉 여부, 건조 기간, 전문 분석 의뢰 상태를 기록한다.

고분 화면에는 “개방 전 기록”과 “개방 후 기록”이 분리되어야 한다. 석실 내부는 개방 순간부터 환경이 바뀌므로, 개방 행위 자체가 기록 대상이다.

## 7. 생산·제철 유적

생산유적은 시설 형태보다 공정 단계가 우선이다.

1. 유적이 어떤 생산 공정에 속하는지 가설을 세운다.
2. 원료, 연료, 시설, 작업면, 송풍, 배출, 부산물, 실패품, 폐기장을 연결한다.
3. 제철유적은 채광, 선광, 배소, 제련, 단야, 용해, 초강 단계 중 어디인지 기록한다.
4. 슬래그, 노벽, 송풍관, 철괴, 목탄은 유물 카드와 분석 시료 카드를 함께 가진다.
5. 폐기장 규모에 따라 전수 수거인지 통계적 표본 수거인지 결정하고 근거를 남긴다.
6. 분석 담당자와 시료 선택 이유를 현장 단계에서 기록한다.

생산유적 화면은 “공정 흐름도 + 시설 + 부산물 + 시료”가 한 묶음으로 보여야 한다.

## 8. 야장 구현으로 이어지는 원칙

- 관찰 사실, 해석, 해석 근거, 불확실성은 모든 카드의 공통 필드다.
- 사진·도면·시료·수습품은 독립 목록이 아니라 원 관찰 기록에 연결한다.
- 유적 유형별 화면은 달라도 기록 ID와 상태값은 공통으로 유지한다.
- 현장 판단이 바뀌는 과정을 지우지 않고 이력으로 남긴다.
- 보고서 문장을 만들 때 원 관찰 기록까지 역추적할 수 있어야 한다.
